This application was made by Hicham Garoum during the learning of Python and the Tkinter library.
This a Python program that uses the Tkinter library to create a graphical user interface (GUI) for an Operations Management System, here for vessel inspection operations during my duty
The application gives the possibility to collect all the information of the inspection's operations and make a database in your local machine by using SQLite.
the mean file is system.py and you can use it directly or by using the login page with login.py.
